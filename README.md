# Melbourne-Housing-Market-Analysis-and-Price-Trend-Exploration
This project performs an in-depth Exploratory Data Analysis (EDA) on the Melbourne Housing Market dataset using Python. The objective is to understand the factors that influence property prices and discover meaningful patterns through data cleaning, visualization, and statistical analysis.

The project helps buyers, sellers, investors, and real estate analysts understand how different property characteristics impact housing prices.

### Objective

TThe main objective of this project is to analyze Melbourne housing market data to identify the major factors affecting property prices. Through exploratory data analysis, the project aims to discover trends, relationships, and patterns among various housing features without applying machine learning models.

### This project focuses on:
- Cleaning and preparing raw housing data
- Handling missing values and duplicate records
- Performing feature engineering
- Visualizing trends using charts
- Finding relationships between housing features and prices
- Generating business insights and recommendations

  ### Project Workflow
**1. Data Collection**  
- Imported the Melbourne Housing Dataset
- Loaded data using Pandas

**2. Data Understanding**
- Explored dataset structure
- Checked data types
- Examined dataset dimensions
- Generated descriptive statistics

**3. Data Cleaning**
- Identified missing values
- Removed duplicate records
- Filled missing numerical values using Median
- Filled missing categorical values using Mode
- Verified cleaned dataset

**4. Exploratory Data Analysis (EDA)**
- Q1.Which Property Type Commands the Highest Average Price?
- Q2.How does the number of rooms affect property prices?
- Q3.Does distance from the city influence property price?
- Q4.How does land size affect property price?
- Q5. Which region has the highest average property price?
- Q6. Are newer houses more expensive than older houses?
- Q7. How does the average property price vary based on the availability of nearby schools and hospitals?
- Q8.Which Suburbs Have the Highest Median Property Prices?

### Key Insights
- Premium suburbs generally have significantly higher property prices.
- Houses typically have higher average prices than units and townhouses.
- Building area and land size show a positive relationship with housing prices.
- Properties located closer to major city areas generally command higher prices.
- Walkable neighborhoods tend to have higher-valued proerties.
- Homes located near schools often have better market value.
- Property condition influences overall pricing.
- Statistical tests confirmed significant price differences across several property categories.
